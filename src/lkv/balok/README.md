## ðŸ”° Pseudocode

```
BEGIN
    DECLARE PANJANG, LEBAR, TINGGI, LUAS_PERMUKAAN, VOLUME AS FLOAT

    INPUT PANJANG
    INPUT LEBAR
    INPUT TINGGI

    SET LUAS_PERMUKAAN TO 2 * ((PANJANG * LEBAR) + (PANJANG * TINGGI) + (LEBAR * TINGGI))
    SET VOLUME TO PANJANG * LEBAR * TINGGI

    OUTPUT LUAS_PERMUKAAN
    OUTPUT VOLUME
END

```
